Splice Samples Sorter

Splice is a cloud-based music creation and collaboration platform which includes a sample library, it is highly used in the world. Unfortunately it comes with a very unpractical folders structure which makes it hard to manually find a specific sample. My app provides a way to rearrange the structure in a more useful way by using a sorting system that sorts all files by categories ready to use.
